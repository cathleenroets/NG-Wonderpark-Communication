# Wonderpark Community Centre Website

A comprehensive community platform designed to connect neighbors, facilitate collaboration, and address local needs through holistic community engagement.

# Features

- Admin Approval System - All community submissions are reviewed before publication to maintain a safe, respectful environment
- Live Chat Support - Real-time communication between community members and centre administrators
- Email Notifications - Automated alerts for new submissions and chat sessions
- Community Announcements - Share news, events, and updates with your neighbors
- Support Requests - Community members can request and offer help
- Needs & Offers Board - Connect people who need assistance with those who can provide it
- Event Management - Promote wellness days, workshops, and community gatherings
- Volunteer Coordination - Facilitate community involvement and skill-sharing
- Mobile Responsive - Fully functional on all devices

## Purpose

This platform serves as a neutral, inclusive hub focused on:

- Holistic community health and wellbeing
- Collaborative problem-solving for local needs
- Building connections across diverse community groups
- Creating safe spaces for engagement and support
- Empowering community-driven initiatives

## Tech Stack

- Frontend: HTML5, CSS3, JavaScript (Vanilla)
- Authentication: Firebase Auth
- Storage: LocalStorage (upgradeable to Firestore/MongoDB)
- Styling: Custom CSS with responsive design
- Icons: Font Awesome

## Quick Start

1. Upload all files to your web hosting
2. Login with default admin credentials (see documentation)
3. Change admin password immediately
4. Configure email notifications
5. Start building your community!
